Sandwich

Sandwich is an academic project in pair realized in C language . The objective was to simulate the behavior of cellular automata. We had to create cells, rules governing their evolution and also set up a management system for these cells in this world. A concrete example of cell evolution in a world is the game of life by John Conway's Game of Life. This project allowed me not only to become more comfortable in the use of C language, but also to discover and use programming tools such as GIT or the Makefile; programming techniques such as separate compilation. It also has a pretext of introduction to SDL which allowed me to use a graphical interface.
